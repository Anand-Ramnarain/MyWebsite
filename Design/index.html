    Style Guide

    My Goal And Aesthetic:

    My goal for my website is to give my users an experience that has easy navigation with an effective interface so that the user can seamlessly move through the website to where they want to go and see what is important on the website without getting distracted.

    Design Elements

    Information Structure - I want to make the information structure clear and simple by spreading my information across multiple pages thus the user can select the page they see the necessary information.

    User Flow- My user's movement is not technically set in chronological order. My user has the freedom to explore my different web pages by choosing which page they would like to view and view or what content they would like to look at first. Allowing the website to be more user-friendly as they have to option to choose where to go.

    Interface Elements- I am using hyperlink controls more than any of the others, my website is simple and requires simple and minimal interface elements so that the user experience is simple, clear and intuitive. Having too many interface components will just overwhelm me as getting that into the website always gets complicated, thus it can make my website unmanageable to complete

    Colour - The colour scheme of black, white, and soft blue I used to create a sleek and modern aesthetic for a website. This allows for timeless appearance. The combination of black and white is classic and timeless, evoking a sense of refinement so that the user can feel as if they are going to get the necessary information. This combined with a soft blue colour creates an overall effect that is both calming. While also keeping a Minimalism look, using a limited colour palette of black, white, and soft blue helped in creating a minimalist design that focuses on simplicity and functionality. This can be a good choice for websites that want to convey a sense of efficiency.Overall, the style of the website using black, white, and soft blue colours aids in creating a versatile and stylish foundation for the website.

    Smooth Transitions -The smooth transition allows to Improve user experience as the Smooth transitions make the website feel more intuitive and seamless, which improves the user experience. When pages transition smoothly, it is easier for users to navigate and explore the site. Thus, allowing engagement from the Smooth transitions increases user engagement by keeping visitors on the site longer. When pages transition smoothly, it can encourage the user to explore more content and stay engaged with the site. To give a Visual appeal, as the smooth transitions are visually appealing and make a website feel more dynamic and interesting. When the pages transition smoothly, it creates a sense of movement and animation that adds visual interest to the site.Overall, using smooth transitions will improve the user experience, increase engagement, and add visual interest.

This will help users to focus on the content and information they are looking for, rather than getting distracted by unnecessary or confusing interface elements. It will improve loading speed of the website, by reducing the number of interactive elements on my website, the website can also load more quickly, which can be a significant advantage for users who may not have high-speed internet connections or who are accessing the site on mobile devices. Thus, making is Mobile-friendly, as keeping the website with a minimal interactive interface makes the website more mobile-friendly, as it was easier to design a responsive layout that works well on smaller screens and large screens when there are fewer interactive elements to consider.
